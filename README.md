# Домашнее задание ШРИ: Инфраструктура

## Принцип работы релизного скрипта

При пуше релизного тега в формате 'v[0-9]+.[0-9]+.[0-9]+*' происходит запуск скрипта **.github/workflows/release.yml**. В нем происходит запуск **release.sh**.
В нем определяются базовые переменные и подключаются скрипты:
- **searchTask.sh** - поиск задачи в реестре
- **addTask.sh** - добаление задачи в реестре
- **updateTask.sh** - обновление задачи в реестре
- **addComment.sh** - добавление комментария к задаче в реестре
- **runTests.sh** - запуск тестов, сохранеие результатов в комментарии
- **artifact.sh** - создание Docker-образа, сохранеие результатов в комментарии
